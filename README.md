# Discord-Bot-Dashboard
A discord bot dashboard with basic template only 
## Features
Welcome Settings 
Invite logs 
more coming soon
## Setup

Rename the ``.env.example to .env``
```npm install```
```node bot.js```
## callback
Goto Application Developer portal And just add auth 
```http://localhost:3000/callback```
